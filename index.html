Interactive Key Technologies

Step through the typical operations of each technology and watch the internal mechanics — replication, partitioning, consensus, indexing. Built for FAANG EM system-design prep.

Each page lets you run CRUD-style operations and see what happens inside: how data is written, synchronized, and queried. Click a technology to start. Pairs with the hi-knowledge/ notes and the system-design-quiz.html self-test.

Key Technologies

🟥
Redis
Primary→replica async replication, SET/GET/DEL, stale replica reads, failover dropping unacked writes.
🟧
Kafka
Produce by key → partition (murmur2), append-only log + offsets, consumer groups, replay.
🔎
Elasticsearch
Index a document → tokenize → inverted index; full-text search vs the SQL LIKE scan.
🟦
Cassandra
Write path: commit log + memtable → SSTable; partition ring; QUORUM reads/writes; tombstones.
⚡
DynamoDB
Partition key → partition; PutItem/GetItem; eventual vs strong (ConsistentRead); GSI.
🐘
PostgreSQL
B-tree index lookup vs full scan; transactions (BEGIN/COMMIT/ROLLBACK); MVCC on update.
🌊
Flink
Event stream → tumbling window aggregation with managed state; watermarks for late events.
🦓
ZooKeeper
Znode tree, ephemeral nodes, leader election, watches firing on change.
🚪
API Gateway
Request → auth → rate limit → route to a backend service → response, with logging.

Data Structures

Step through each structure's operations and watch the internal state — for the Code Comprehension & Systems Architecture rounds.

📊
Arrays
Indexed cells, amortized-O(1) append with capacity doubling, O(n) mid insert/delete.
🔗
Linked Lists
Pointer rewiring, head/tail/given-node ops, Floyd cycle detection.
🥞
Stacks
LIFO push/pop/peek + a valid-parentheses walkthrough.
🎟️
Queues & Ring Buffer
Ring buffer with wrap-around head/tail; reject vs overwrite-oldest.
🗂️
Hash Tables
Hash → bucket, chaining collisions, load factor, resize + rehash.
⛏️
Heaps
Tree + array views, sift-up/down, min/max, top-K demo.
🌲
Sorted Sets / BST
BST insert/search/range; watch sorted inserts degenerate to a chain.
🪜
Skip Lists
Express lanes, coin-flip promotion, skip-right/drop-down search (Redis ZSET).
🌿
Tries
Prefix tree: insert, startsWith, autocomplete — O(L) regardless of size.
🗃️
B-Trees / B+Trees
High fan-out, split-on-overflow, linked-leaf range scans (DB indexes).
🕸️
Graphs
Adjacency list; BFS (queue), DFS (stack), topological sort.
🧩
Union-Find
Union by rank + path compression; component count; cycle detection.
♻️
LRU / LFU Cache
Hash map + doubly linked list, O(1) get/put, eviction (LRU & LFU).
🌸
Bloom Filters
Bit array + k hashes; definitely-not vs possibly-yes; false positives.
